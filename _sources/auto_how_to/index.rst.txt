How-to guides
=============

Runnable recipes grouped by topic. Every script here is executed at
documentation build time — what you see rendered is exactly the output
it produced on the build machine. Download the ``.py`` or ``.ipynb`` of
any example from the link at the bottom of each page.

All scripts assume an OpenFOAM environment has been sourced and operate
on the cases shipped under ``examples/`` — no external data is required.
